Podłącz swój system klimatyzacji Mitsubishi Heavy Industries do Homey.

Włącz lub wyłącz swoje urządzenie HVAC, ustaw temperaturę i dostosuj tryb — wszystko za pomocą technologii podczerwieni. Nie potrzebujesz osobnego pilota; wszystko jest zintegrowane.

Ta aplikacja pozwala Homey Bridge i Homey Pro kontrolować jednostki klimatyzacyjne Mitsubishi Heavy Industries za pomocą podczerwieni, dzięki czemu możesz je kontrolować i automatyzować w aplikacji Homey. Homey Pro lub Homey Bridge muszą znajdować się w tym samym pomieszczeniu co system klimatyzacji, ponieważ sygnały podczerwieni nie przechodzą przez ściany.
